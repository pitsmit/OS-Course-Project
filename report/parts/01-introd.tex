\chapter*{ВВЕДЕНИЕ}
\addcontentsline{toc}{chapter}{ВВЕДЕНИЕ}

Предотвращение утечек данных через USB-устройства является одной из важнейших
задач Data Leak Prevention программно-аппаратных комплексов~\cite{WhatIsDLP}. 
Основными механизмами для решения этой задачи в ядре Linux являются драйверы 
устройств и перехват функций ядра. Данная работа посвящена созданию 
и исследованию загружаемого модуля ядра для подмены данных, 
передаваемых на USB-накопители, с использованием перехвата 
управления у функций.
